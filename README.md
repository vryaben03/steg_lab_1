# steg_labs
A steg study

## Команды для запуска

```bash
py stego.py                    # Задание 1: LSB-стеганография, визуализация битовых плоскостей, метрики
py additional_task.py          # Задание 1: доверительные интервалы для PSNR и дисперсии
py watermarking.py             # Задание 2: тест двух методов внедрения ЦВЗ (LSB с ключом и адаптивный)
py watermark_research.py       # Задание 2: исследовательская часть - доверительные интервалы PSNR
py prepare_dataset.py          # Подготовка медицинских изображений (JPG → PGM, 512×512)
py prepare_texture.py          # Подготовка текстурных изображений (JPG → PGM, 512×512)
